# Hovercards

Show previews of your posts on hover.

![](preview.gif)

## Installation

1. Clone this repository
2. `npm install`
3. `npm run build`
4. Activate the plugin.

## Development

1. Clone this repository.
2. `npm install`
3. Add `define( 'HOVERCARDS_DEV', true )` to your wp-config.
4. Activate the plugin.
5. `npm start`
